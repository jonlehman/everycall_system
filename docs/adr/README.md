# ADR Index

Use ADRs (Architecture Decision Records) to capture irreversible or expensive-to-change decisions.

## Status values
- `proposed`
- `accepted`
- `superseded`
- `deprecated`

## Naming
- File format: `NNNN-short-title.md`
- Example: `0002-event-bus-contracts.md`

## Required sections
- Context
- Decision
- Consequences
- Alternatives considered
